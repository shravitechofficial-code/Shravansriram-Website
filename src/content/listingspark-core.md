# ListingSpark Core
ListingSpark Core is an AI-powered web application to help e-commerce sellers generate a foundational set of three product listing images: Key Benefits Showcase, How-To-Use/Instructions, and Key Ingredients/Composition. Users can edit AI-generated prompts and regenerate images for customization.
